SIETE Y MEDIA
-------------------

Juego Siete y Media realizado en PHP 

-- Programa Requerido --

-> PHP


-- Como acceder al directorio --

-> Desde el terminal poner lo siguiente: cd /SieteyMedia


-- Ejecución del programa --

-> php sieteymedia.php


El programa automáticamente se iniciará cuando la puntuación del jugador sea menor que la de la máquina. En este caso desde consola
pregunta si queremos empezar.
Al finalizar la partida nos preguntará el programa si deseamos volver a jugar.
